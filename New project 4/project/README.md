# Personal Portfolio Website

A modern, responsive, recruiter-friendly personal portfolio website for a Diploma Computer Science Engineering student. It is built with plain HTML, CSS, and JavaScript, so it can be opened directly in a browser or deployed to GitHub Pages.

## Features

- Responsive desktop and mobile layout
- Sticky shrinking navbar
- Smooth scrolling navigation
- Full-screen preloader with fade-out
- Scroll reveal animations
- Project, certification, experience, skills, about, and contact sections
- Optimized image usage with lazy loading and alt text
- GitHub Pages-ready static structure

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript

## Setup Instructions

1. Open `index.html` in your browser.
2. Replace `Replace with user name` with your real name in `index.html`.
3. Replace placeholder links such as `https://github.com/your-username` and LinkedIn links with your real profiles.
4. Add your images inside the `images` folder:
   - `profile.jpg`
   - `project1.jpg`
   - `project2.jpg`
   - `project3.jpg`
5. Update the contact form action URL in `index.html` if you use Formspree or your own backend.

## What To Customize

- Name: Search for `Replace with user name` in `index.html`.
- GitHub username: Search for `your-username`.
- LinkedIn profile: Search for `your-linkedin-profile`.
- Contact form: Search for `your-form-id`.
- Project descriptions and live demo links: Update each project card in the Projects section.
- Certifications: Edit certification names and descriptions in the Certifications section.

## GitHub Pages Deployment

1. Create a new GitHub repository.
2. Upload the contents of this `project` folder to the repository.
3. Commit and push your files.
4. Go to repository `Settings`.
5. Open `Pages`.
6. Under `Build and deployment`, choose `Deploy from a branch`.
7. Select the `main` branch and `/root` folder.
8. Save the settings.
9. Your portfolio will be available at:

```text
https://your-username.github.io/repository-name/
```
